# Desktop Launchers

Canonical desktop launcher backups for BloomCore Pi bench apps.

These files are copied to:

- `~/Desktop/`
- `~/.local/share/applications/`

Current launchers:

- `SkyCam.desktop`
- `HarnessMap.desktop`
- `BloomRestore.desktop`
- `BloomDoctor.desktop`

Notes:

- Paths currently assume Raspberry Pi user `quarterbitgames`.
- Keep these launchers simple and readable.
- If a desktop icon breaks, restore from this folder.
