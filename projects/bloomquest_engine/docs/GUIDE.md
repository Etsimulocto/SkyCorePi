# BloomQuest Guide

## Directions

Choose a layer, choose a premade part, and click the grid to place it. Click a placed part again to edit its fields. Press Apply Changes and save with Ctrl+S. Place one Player, then press F5 to test the room.

Use the Rooms menu to create, duplicate, or open rooms. A Door connects rooms using Target Room, Target X, and Target Y.

## Terms

Part: a reusable game piece.

Room: one saved map.

Layer: a category of parts.

Property: an editable field.

Action: a premade behavior.

Counter: a number such as health, coins, score, or keys.

Solid: blocks movement.

Play Mode: a temporary test run.
